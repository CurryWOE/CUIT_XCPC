# 最短路
多起点多终点可以建源点汇点，点权变成连向源点汇点的边权，然后由汇点做最短路

dij做最短路计数不能处理边权0的情况
## 最短路径树
dijkstra，维护一个长度 n 的数组，在每个点进队列时记录这个点的入边编号，整个流程跑完后即得最短路径树的边

如果要求所有最短路径树中，边权和最小值，在遍历边时，若dis变小，则该入边记录一下，若dis相等，则记录min(之前记录的最小入边权，当前入边权)，整个流程跑完后即得“所有最短路径树中，边权和最小值”。边权和最大值反之亦然
```c++
namespace dijkstra//O(mlog m)
{
    struct edge
    {
        int v;
        ll w;
        bool operator>(const edge &b)const{return w>b.w;}
    };
    int n,vis[N];
    ll dis[N];
    vector<edge> e[N];
    priority_queue<edge,vector<edge>,greater<edge> > q;
    void add_edge(int u,int v,ll w)
    {
        e[u].push_back({v,w});
        // e[v].push_back({u,w});
    }
    void run(int begin)
    {
        memset(dis,0x3f,sizeof(dis[0])*(n+1));
        memset(vis,0,sizeof(int)*(n+1));
        dis[begin]=0;
        q.push({begin,0});
        while(!q.empty())
        {
            int u=q.top().v;
            q.pop();
            if(vis[u])
                continue;
            vis[u]=1;
            for(edge &ed:e[u])
            {
                int v=ed.v;
                if(dis[v]<=dis[u]+ed.w)
                    continue;
                dis[v]=dis[u]+ed.w;
                q.push({v,dis[v]});
            }
        }
    }
    void init(int nn)
    {
        n=nn;
        for(int i=0;i<=n;++i)
            e[i].clear();
    }
};
```
