# 字符串哈希
假设有一个超级大的玩具房，当你想要找一个玩具，最简单的做法是一个一个看，直到找到想要的那个玩具

但是这个做法太慢了，如果有一个工具能快速告诉你，你想要的玩具在什么位置就好了

哈希就是这样一个工具

这里引入几个概念：

键：玩具的名字

哈希值：玩具所在的位置

哈希函数：转换器，输入想要的玩具的名字，输出玩具所在的位置
# 字符串哈希
## 键
字符串
## 哈希值
非负整数，并通过取模限制大小

当字符串很长的时候，如果不限制哈希值大小，那么哈希值就会很大，没法用c++里的基础类型存储

解决办法是用一个大质数取模
## 冲突
因为取模，两个不同的字符串计算出来的哈希值可能一样

不同输入相同输出称之为冲突

巧妙设计哈希函数来尽量避免冲突
## 哈希函数
这里以前缀哈希为例，并假设字符串S只由小写字母组成，设P的值为131，设字符串长度为len

$字符串S的哈希值=(P^{len-1}\times(S_0-'a'+1)+P^{len-2}\times(S_1-'a'+1)+...+P^{0}\times(S_{len-1}-'a'+1))\%mod$

举个例子：字符串"a"的哈希值是1，字符串"aa"的哈希值是132=131+1，字符串"aaa"的哈希值是 $17293=131^2+131+1$
### 为什么这么设计？
1. $S_0-'a'+1$ ：
   * 大于0，小于P
   * 如果可以等于0，比如设计成$S_0-'a'$，那么字符串"a","aa","aaa"的哈希值会一样
   * 如果可以等于P，比如设计成P=3，那么字符串"aa","d"的哈希值会一样
   * 如果是大写字母组成的字符串，换成 $S_0-'A'+1$
   * 如果是其他字符串，自己设计一下满足大于0小于P即可

2. 模数mod：
   * $mod\times (mod-1)<unsigned\ long\ long的最大值$，防止溢出造成的冲突
   * mod是大质数，为了避免冲突
   * mod不可以用 $2^{64}$，也就是自然溢出，随机数据都容易卡掉
   * 常用的mod有998244353，$10^9+7$

3. 底数P：
   * 小于mod，大于 $S_0-'a'+1$
   * 常用的P有131，13331

4. 为什么是有规律的乘积之和
   * 本质上是把字符串当成P进制数，可以快速计算子串的哈希值，拼接字符串的哈希值
   * 预处理前缀哈希值，可以快速计算某个子串的哈希值
   * 如果已知字符串 $S_0$ , $S_1$ 的哈希值，$S_1$ 的长度，那么可以快速计算拼起来的字符串 $S_0 + S_1$ 的哈希值
   * 即 $Hash_{S_0}\times P^{len_{S_1}}+Hash_{S_1}$
### 如果这么设计之后还是冲突怎么办？
理论上是扩大哈希值大小，不再是0到mod-1，可以通过中国剩余定理证明最大哈希值是模数之积-1

实际做法是用不同模数计算多个哈希值，只有两个不同字符串，多个哈希值都相同，才算冲突

在竞赛里一般是，双模数算两个哈希值，更多模数或单模数很少见
## 字符串哈希的用途
$O(1)$ 比较两个字符串是否相同

当哈希值不同，两个字符串绝对不同

当哈希值相同，两个字符串可能相同。  
因为精心设计了避免冲突的方法，所以冲突的概率很小，因此我们认为哈希值相同就代表字符串相同
### 判断回文串
如果把字符串倒过来，再哈希一遍，如果哈希值相同，代表这个字符串是回文串
## 建模小技巧
字符无关，位置有关，转化为26个01串
# 实现
```cpp
constexpr uint32_t N=1e6+1,modnum=2;
constexpr uint64_t base=131,mod[modnum]={(uint64_t)1e8+7,(uint64_t)1e9+9};
uint64_t pw[modnum][N];
void init()
{
    for(uint32_t j=0;j<modnum;++j)
    {
        pw[j][0]=1;
        for(uint32_t i=1;i<N;++i)
            pw[j][i]=pw[j][i-1]*base%mod[j];
    }
}
void initHash(const string &s,array<vector<uint64_t>,modnum> &h)
{
    for(auto &vec : h)
        vec.assign(s.size() + 1, 0);
    for(uint32_t j=0;j<modnum;++j)
        for(uint32_t i=0;i<s.size();++i)
            h[j][i + 1] = (h[j][i] * base + s[i] - 'a' + 1) % mod[j];
}
array<uint64_t,modnum> subHash(const array<vector<uint64_t>,modnum> &h,uint32_t l,uint32_t r)
{
    array<uint64_t,modnum> res;
    for(uint32_t i=0;i<modnum;++i)
        res[i]=(h[i][r]+mod[i]-h[i][l-1]*pw[i][r-l+1]%mod[i])%mod[i];
    return res;
}
```
# 判断回文串
```cpp
void initHash(const string &s,array<vector<uint64_t>,modnum> &rev)
{
    for(auto &vec : rev)
        vec.assign(s.size() + 1, 0);
    for(uint32_t j=0;j<modnum;++j)
        for(uint32_t i=0;i<s.size();++i)
            rev[j][i+1]=(rev[j][i]*base+s[s.size()-1-i]-'a'+1)%mod[j];
}
array<uint64_t,modnum> subHash(const array<vector<uint64_t>,modnum> &rev,uint32_t l,uint32_t r)
{
    array<uint64_t,modnum> res;
    for(uint32_t i=0;i<modnum;++i)
        res[i]=(rev[i][rev[0].size()-l]+mod[i]-rev[i][rev[0].size()-1-r]*pw[i][r-l+1]%mod[i])%mod[i];
    return res;
}
```
