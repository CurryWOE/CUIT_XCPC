# 虚树
## 问题特征
* 多次询问，每次询问给出一个点集 S，点集大小 |S| 总和与树点数同阶
* 查询信息仅和点集以及点两两间路径相关
## 从暴力到虚树
暴力做法是遍历整颗树的同时处理信息

考虑新建一个树，保持点集之间的拓扑关系，但是又尽可能压缩点数，使得查询复杂度和新树点数相关，和原树点数无关

不在点集也不在点两两间路径，的点，对查询无关，不需要加入新树

路径上的点数太多了，如果直接维护点两两间路径，有 |S|(|S|-1)/2 条路径，太多，考虑中间点来维护路径

考虑菊花图，显然LCA能以最少点保持拓扑关系，实际上，一般图也是，先给出结论：一颗树上选出点集 S，它们两两间不同的LCA数量最多 |S| -1 个

简单的证明：要把 k 个独立的点最终合并成一个大连通块，你只需要进行 k-1 次合并

因此新树的点，是询问给出的点集，以及点集两两间不同的LCA，最多 2|S|-1 个点

建完新树后，在新树上处理信息

下面的实现中，由于加边顺序是从虚树叶子到虚树根的拓扑序，如果能在加边的时候计算中间答案，即可删去深度大的点的边，否则记录有边的点。
## 建树方法1：两次排序
将点集按DFS序排序，相邻两点的LCA，包括了任意两点的LCA，这样获得了新树的点集

将新树点集按DFS序排序，由于上一步可能有重复LCA需去重，相邻两点的LCA连向dfs序大的点，这样就获得了新树的边集

[洛谷模板题两次排序速度第一](https://www.luogu.com.cn/record/277443172)545ms
```cpp
uint32_t dfn[N],s[N*2];
void buildVirtualTree(uint32_t n)
{
    auto cmp=[](int x,int y){ return dfn[x]<dfn[y]; }
    sort(s,s+n,cmp);
    for(uint32_t i=1;i<n;++i)
        s[n+i-1]=LCA(s[i],s[i-1]);
    sort(s,s+n*2-1,cmp);
    n=unique(s,s+n*2-1)-s;
    for(uint32_t i=n-1;i;--i)
        addEdge(LCA(s[i-1],s[i]),s[i]);
    for(uint32_t i=0;i<n;++i)
        clearEdge(s[i]);
    return calculate(s[0]);
}
```
## 建树方法2：单调栈
维护一个单调栈，深度单调递增，且满足栈中任意两点在原树上是祖先子孙关系，出栈时栈次顶连边向栈顶

将点集按DFS序排升序。先将第一个点插入单调栈，主要是简化代码。

然后遍历点集，考虑当前点和栈顶的LCA，它有两种可能，

一种是LCA是栈顶，那么当前点入栈

一种是LCA是栈顶的祖先，那么不断出栈，直至栈顶深度大于LCA，栈次顶深度小于LCA，此时LCA入栈（也有可能是LCA已在栈中，那么LCA不重复入栈），最后将当前点入栈

最后不断出栈，直至栈只剩一个点，该点为虚树的根

下面的实现中，祖先子孙的深度大小比较，能通过DFS序比较，所以省去深度数组

[洛谷模板题速度第一](https://www.luogu.com.cn/record/277454285)378ms
```cpp
uint32_t dfn[N],s[N],stk[N];
auto buildVirtualTree(uint32_t n,bool compute_while_adding)
{
    sort(s,s+n,[](uint32_t x,uint32_t y){ return dfn[x]<dfn[y]; });
    stk[1]=s[0];
    uint32_t top=1;
    vector<uint32_t> pointHaveEdge;
    pointHaveEdge.reserve(n);
    for(uint32_t i=1;i<n;++i)
    {
        auto lca=LCA(s[i],stk[top]);
        for(;top>1 && dfn[lca]<=dfn[stk[top-1]];--top)
        {
            addEdge(stk[top-1],stk[top]);
            if(compute_while_adding)
                clearEdge(stk[top]);
            else
                pointHaveEdge.push_back(stk[top-1]);
        }
        if(lca!=stk[top])
        {
            addEdge(lca,stk[top]);
            if(compute_while_adding)
                clearEdge(stk[top]);
            else
                pointHaveEdge.push_back(lca);
            stk[top]=lca;
        }
        stk[++top]=s[i];
    }
    for(;top>1;--top)
    {
        addEdge(stk[top-1],stk[top]);
        if(compute_while_adding)
            clearEdge(stk[top]);
        else
            pointHaveEdge.push_back(stk[top-1]);
    }
    if(compute_while_adding)
    {
        auto ans=calculate(stk[1]);
        clearEdge(stk[1]);
        return ans;
    }
    else
    {
        auto ans=dfs(stk[1]);
        for(auto u:pointHaveEdge)
            clearEdge(u);
        return ans;
    }
}
```
