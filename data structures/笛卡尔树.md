# 笛卡尔树(Cartesian Tree)
一种二叉树，每个点有两个属性，键和值，只考虑键，树是二叉搜索树，只考虑值，树是堆

当键键互不相同，值值互不相同，笛卡尔树结构唯一

对于一个数组，下标为键，元素为值，构造出的笛卡尔树满足，树上点 l 和点 r 的LCA的值等于区间 [l,r] 最值

如果把键和优先级随机化，笛卡尔树就是一种平衡树--Treap
```cpp
template <typename K, typename V>
class CartesianTree
{
public:
    struct DataNode
    {
        K key;
        V val;
        uint32_t l, r;
        DataNode() : key(K()), val(V()), l(0), r(0) {}
    };
    std::vector<DataNode> tree;
    uint32_t root;
    void build(std::vector<std::pair<K, V>>& data)
    {
        sort(data.begin(), data.end(), [](const auto& a, const auto& b) {
            return a.first < b.first; 
        });
        tree.assign(data.size()+1,DataNode());
        for(int i=0;i<data.size();++i)
        {
            tree[i+1].key=data[i].first;
            tree[i+1].val=data[i].second;
        }
        std::vector<uint32_t> stk;
        stk.reserve(data.size());
        for (int i = 1; i <= data.size(); ++i)
        {
            uint32_t last_popped = 0;
            // 根据 Value 维护小根堆单调性
            while (!stk.empty() && tree[stk.back()].val > tree[i].val)
            {
                last_popped = stk.back();
                stk.pop_back();
            }
            tree[i].l = last_popped;
            if (!stk.empty())
                tree[stk.back()].r = i;
            stk.push_back(i);
        }
        root = stk.empty() ? 0 : stk.front();
    }
};
```
