# 树的直径
树上任意两节点之间最长的简单路径即为树的直径。从定义上讲更像是“最大子段和”在树上的推广

边权非负的树，任意点作为起点搜索出最远点，再以最远点作为起点搜索出第二轮最远点，二点距离即为树的直径

存在负边权的树，维护u作为端点的最长链，考虑其不同儿子拼出来最长链
```cpp
int dp[N];
void diameter(uint32_t u, uint32_t fa, int &res)
{
    for (auto v : e[u])
    {
        if (v == fa)
            continue;
        diameter(v, u, res);
        res = max(res, dp[u] + dp[v] + w(u,v));
        dp[u] = max(dp[u], dp[v] + w(u,v));
    }
}
```
