# 静态开点
定义：所有要表示的区间，其节点下标已经确定
## 节点编码方案的形式化描述
- 根节点对应区间 $[1,n]$。
- 若节点对应区间 $[l,r]$ 且 $l<r$，令 $mid = \lfloor\frac{l+r}2\rfloor$，则该节点有左子节点对应区间 $[l,mid]$，右子节点对应区间 $[mid+1,r]$。
- 若 $l=r$，则该节点为叶节点。

>如果想把区间拓展到非正数，看算出来的下标是否是非正数，如果是非正数，加偏移即可
### 方案一：堆式编码
- 根节点下标为 $1$
- 若某非叶节点下标为 $v$，则其左子节点下标为 $2v$，右子节点下标为 $2v+1$
- 最大下标可通过下列函数计算，粗上限为 $f(n)\le 4n$
```cpp
uint32_t f(uint32_t n)
{
    if(has_single_bit(n))
        return 2*n-1;
    auto k=bit_width(n);
    auto r=bit_width(n^(1u<<(k-1)));
    return (((1u << r) - 1) << (k - r + 1)) | 1; 
}
```
上述函数的证明如下：

根深度为0，深度为 d 的节点编号范围是：$[2^d, 2^{d+1})$，因此最深点编号最大

因为右儿子编号比左儿子大，所以最深点里面的最右点编号最大

对于叶子点数 n 为2的幂，线段树是满二叉树，最大编号即为 $2n-1$

对于非2幂，考虑当前点到最深层的剩余深度，如果往右儿子走还能走到最深深度就走右儿子，否则走左儿子，最后一步一定是走右儿子。设 $n=2^k+r$ ，其中 $r\in[1,2^k)$，那么有 k=bit_width(n)-1，$r=n-2^k$，就是先走右儿子 bit_width(r) 次，再走左儿子 k+1 - bit_width(r) 次，最后走右儿子1次
### 方案二：中点编码
- 根节点下标为 1
- 区间为 $[l,r]$ 的非叶节点，其左儿子下标为 $2mid$，右儿子下标为 $2mid+1$
- 最大下标 $2n-1$

**定理**  
- 所有左子节点的下标恰好为 $\{2,4,6,\dots,2n-2\}$，
- 所有右子节点（含根节点）的下标恰好为 $\{1,3,5,\dots,2n-1\}$。

**引理：不同非叶点的区间中点一定不同，且为[1,n-1]双射**

如果树上非叶两点没有祖先后代关系，在它们的LCA往下走时，区间已无交集，因此不可能有相同区间中点

如果有祖先后代关系，设点A为祖先，那么点B不可能在A的右子树，因为右子树区间无交集，因此不可能有相同区间中点

假设点B在左子树，因为区间中点为m，又因为左子树区间右端点最大m，所以左端点只能是m，左右端点相等是叶节点，因为B不能是叶节点，也就不可能左子树，所以树上非叶两点区间中点一定不同

又因为线段树是完全二叉树，非叶点数量等于叶点数量-1

又因为区间中点范围是 [1,n-1]，所以是双射

所以左儿子下标是 $[1,n-1]*2$，右儿子下标是 $[1,n-1]*2+1$
### 方案三：叶偶编码
- 区间为 $[l,r]$ 的节点，下标为 $(l+r) \mid (l < r)$，其中 $\mid$ 表示按位或运算
- 最大下标 $2n$

易得叶节点下标为偶数且互不相同，非叶节点下标为奇数

假设非叶点区间中点x，那么下标可能是2x或2x+1，而非叶点只能是奇数，那么下标只能是2x+1

因为前文已经证明非叶节点区间中点是 [1,n-1] 的双射，所以非叶点下标互不相同
### 方案四：DFS序
- 最大下标 $2n-1$
### 方案五：叶大编码
- 叶节点下标为 $n+l$，非叶节点下标为 $mid$
- 最大下标 $2n$

# 动态开点
定义：所有要表示的区间，实时分配其节点下标

每个节点新增两个变量l，r，表示其左右儿子的节点下标

不使用0作为实际存在的节点的下标，且信息(sum)为空。若某节点的下标为0，表示尚未分配下标

修改时搜索到尚未分配下标的节点，直接分配一个新的下标

查询时搜索到尚未分配下标的节点，说明没有信息要合并，直接返回

初始化要把所有节点都设置为尚未分配下标
```cpp
namespace SegmentTree
{
    struct node
    {
        int l,r;
        ll sum;
    }st[N];
    int tot;
    void update(int &id,int segl,int segr,int pos, ll val)
    {
        if(!id)
            id=++tot;
        if (segl==segr)
        {
            st[id].sum+=val;
            return;
        }
        int mid = (segl + segr)>>1;
        if (pos <= mid)
            update(st[id].l,segl,mid,pos,val);
        else
            update(st[id].r,mid+1,segr,pos,val);
        st[id].sum=st[st[id].l].sum+st[st[id].r].sum;
    }
    ll query(int id,int segl,int segr,int L, int R)
    {
        if(!id)
            return 0;
        if (L <= segl && segr <= R)
            return st[id].sum;
        int mid = (segl + segr)>>1;
        ll res = 0;
        if (L <= mid)
            res+=query(st[id].l, segl,mid,L, R);
        if (R > mid)
            res+=query(st[id].r,mid+1,segr, L, R);
        return res;
    }
    void init()
    {
        for(;tot;--tot)
            st[tot]={0,0,0ll};
    }
}
st::init();
st::update(1,1,n,pos,x);
st::query(1,1,n,l,r);
```
## 加内存池的动态开点
如果认为一个节点暂时无用，回收该节点的下标，在申请下标时优先使用回收的下标

对于信息是区间和的情况，当节点信息为0时，认为其暂时无用

部分毒瘤题需要此技巧，省空间
```cpp
namespace segmentTreeWithMemoryPool
{
    struct node
    {
        int l,r;
        ll sum;
    }st[N];
    int tot;
    vector<int> memoryPool;
    void newNode(int &id)
    {
        if(memoryPool.empty())
            id=++tot;
        else
        {
            id=memoryPool.back();
            memoryPool.pop_back();
        }
    }
    void delNode(int &id)
    {
        memoryPool.push_back(id);
        id=0;
    }
    void update(int &id, int l, int r,int pos,ll val)
    {
        if(!id)
            newNode(id);
        st[id].sum+=val;
        if(l==r)
            return;
        int mid = (l+r)>>1;
        if (pos <= mid)
            update(st[id].l,l,mid,pos,val);
        else
            update(st[id].r,mid+1,r,pos,val);
        if(st[id].sum==0)
            delNode(id);
    }
    ll query(int id,int segl,int segr,int l, int r)
    {
        if(!id)
            return 0;
        if (l <= segl && segr <= r)
            return st[id].sum;
        int mid = (segl + segr)>>1;
        ll res=0;
        if (l <= mid)
            res+=query(st[id].l,segl,mid,l,r);
        if (r > mid)
            res+=query(st[id].r,mid+1,segr,l,r);
        return res;
    }
}
```
