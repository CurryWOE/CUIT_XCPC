## 归并排序
归并两个有序序列，得到一个有序序列，递推就是任意个有序序列，可以归并成一个有序序列，这个结论在[这个题目](https://codeforces.com/contest/207/problem/A3)很有用

稳定排序，当然也可以通过加原始下标作为最后的关键字来实现稳定

大问题划分成子问题，递归处理子问题，合并子问题的答案求解当前问题的答案，非常多算法的思想，包括快速排序，点分治等

std::stable_sort的实现算法

CDQ分治的前置知识，归并排序的合并过程天然支持二维偏序问题的离线处理

PingPong归并是求逆序对的最快算法，例程如下，[洛谷速度第一](https://www.luogu.com.cn/record/274199246)
```cpp
void PingPongMerge(int l,int r,int* src,int* dest,long long& ans)
{
    if(r-l<=16)
    {
        for (int i = l + 1; i <= r; ++i)
        {
            int v = src[i], j = i;
            for(;j > l && src[j - 1] > v;--j)
                src[j] = src[j - 1];
            src[j] = v;
            ans += (i - j);
        }
        memcpy(dest + l, src + l, (r - l + 1) * sizeof(int));
        return;
    }
    int mid=(l+r)>>1;
    PingPongMerge(l,mid,dest,src,ans);
    PingPongMerge(mid+1,r,dest,src,ans);
    int i=l,j=mid+1,k=l-1;
    while(i<=mid && j<=r)
    {
        if(src[i]<=src[j])
            dest[++k]=src[i++];
        else
        {
            dest[++k]=src[j++];
            ans+=mid-i+1;
        }
    }
    while(i<=mid)
        dest[++k]=src[i++];
    while(j<=r)
        dest[++k]=src[j++];
}
long long solve(vector<int>& a)
{
    vector<int> tmp=a;
    long long ans=0;
    PingPongMerge(0,a.size()-1,a.data(),tmp.data(),ans);
    return ans;
}
```

《算法导论》有详细介绍其实现技巧，包括三数取中值作为基准值等

std::qsort的实现算法，但因为速度不够快，一般不使用std::qsort
## 快速选择
快速排序只递归单边，可以从某个序列中 $O(n)$ 找到第 k 小的元素 Val

std::nth_element的实现算法，以下为函数原型于介绍，摘抄自CPP Reference
```cpp
void nth_element<_RAIter>(_RandomAccessIterator __first, _RandomAccessIterator __nth, _RandomAccessIterator __last)
void nth_element(RandomAccessIterator first, RandomAccessIterator nth, RandomAccessIterator last, Compare comp);
```
该函数可以从某个序列中 $O(n)$ 找到第 k 小的元素 Val，并将 Val 移动到序列中第 k 的位置处。不仅如此，整个序列经过 nth_element() 函数处理后，所有位于 k 之前的元素都比 Val 小，所有位于 k 之后的元素都比 Val 大，在默认的升序排序规则（std::less）的条件下
## 内省排序
最通用的排序，结合了插入排序，堆排序，快速排序，最坏复杂度 $O(n\log n)$

std::sort的实现算法，通过内联优化，三种排序结合，不丢弃类型信息，模板函数，比std::qsort快了很多倍

std::sort的比较函数必须满足严格弱序，笔者曾在某场牛客多校赛被坑了一整场，例如：使用 `<=` 而非 `<` 可能导致越界或死循环

# 用到排序的技巧
## 赋予序列单调性
当顺序不影响结果，有序序列上处理问题可能比无序序列，简单很多

比较有名的算法包括双指针，单调栈，单调队列，扫描线等
## 离散化
当只关注数字之间的相对大小关系，而不关注具体数值。使用此方法可将值域缩小成不重复数字的个数

排序，去重，映射
## 结构体多关键字
根据多个属性确定优先级，非常多题目使用此技巧

比较有名的算法是莫队及其奇偶化排序优化
## 多维偏序
对第一维排序消除第一维限制，剩下的维度可通过数据结构或分治解决

比较有名的算法比如排序+树状数组/线段树解决二维偏序，CDQ分治解决三维偏序
