# 等额取石问题
X堆石头，每堆至少1个石头，每次从Y堆(Y>1)中取石头，每堆取走的石头数量一样，单人游戏，问能取完的条件，最多操作次数及取法，最少次数的最小值最大值，最少次数及取法
# 每次操作的Y任意
## 合法条件
$X\ge2$，且最大堆不超过其余堆的总和
## 证明
不满足这两个条件的情况显然不能取完，不做证明

下面证明了满足这两个条件的情况都可以使用下面的构造性取法

取法：每堆取走的石头固定为1。如果最大堆大于1，每次选取最大的两堆去取，否则一次取所有的非空堆。

证明：设操作前的最大堆为 $A_1$，次大堆为 $A_2$，总数位 $S$。从 $A_1$ 和 $A_2$ 中各拿走 1 颗石头。新的总数变为：$S' = S - 2$。原来的两堆变成：$A_1 - 1$ 和 $A_2 - 1$

首先最终情况满足条件2，下面要证明在取完前时刻满足条件1和2

1. 如果 $A_1>A_3$

则新局面的最大堆 $A_1' = A_1 - 1$。要证明新局面依然满足 $2A_1' \le S'$：$$2A_1' = 2(A_1 - 1) = 2A_1 - 2$$因为已知原有条件 $2A_1 \le S$，所以：$$2A_1 - 2 \le S - 2 = S'$$从而得出：$2A_1' \le S'$。条件依然成立

2. 如果 $A_1=A_2=A_3$

则新局面的最大堆 $A_1' = A_3$。要证明 $2A_3 \le S'$：$$2A_3 = A_2 + A_3$$而总数 $S = A_1 + A_2 + A_3 + \dots \ge A_1 + A_2 + A_3$。因为 $A_1 \ge 1$（至少有石头才能操作），所以：$$S - 2 \ge (A_1 - 2) + A_2 + A_3$$

* 2.1 如果 $A_1 \ge 2$
    
那么 $A_1 - 2 \ge 0$，所以：$$S' = S - 2 \ge A_2 + A_3 = 2A_3 = 2A_1'$$从而得出：$2A_1' \le S'$。条件依然成立

* 2.2 如果 $A_1 = 1$

一次性取完即可

至此已证明完时刻满足条件2，下面证明时刻满足条件1

1. 若最大堆大于1，取最大的两堆，最大堆不会变为0，因此每次操作堆数最多减1，因此从3堆变化最大也就是2堆

2. 若最大堆等于1，此时全部取完，直接结束
## 最多次数
$\lfloor\frac S2\rfloor$，取法：每次去取最大的两堆，每堆取一个石头，最后一次操作可能要取3堆，每堆取一个石头

证明：

1. 若最大堆大于1，取最大的两堆，最大堆不会变为0，因此每次操作堆数最多减1，因此从多堆只能变成2堆而不是1堆
* 1.1. 根据上面证明的“最大堆不超过其余堆的总和”时刻满足，当只有2堆时，要满足这个条件，两堆只能相等
2. 若最大堆等于1
* 2.1. 若堆数大于3，只可能变成2堆或3堆的情况
* 2.2. 2堆或3堆都只能1次取完
## 最少次数
最小值1，最大值 $X-1$，每次操作至少清空一个堆，形如[1,2,4,8,15]的数据，每次至多清空一个堆

最少次数NP问题，DP
# 每次操作的Y固定
## 合法条件
$X\ge Y$，且 $S$ 是 $Y$ 的倍数，且最大堆不超过 $\frac SY$
## 证明
不满足这三个条件的情况显然不能取完，不做证明

其余证明见最多次数
## 最多次数
$\frac SY$，每次取前Y大堆，每堆取1个石头。证明类似Y=2的情况
## 最少次数
最小值1，最少次数最大值 $\frac SY$，当所有堆只有一个石头时取到

当 $Y\ge 3$，NPH问题，整数线性规划

当 $Y=2$ ，贪心取最大的两堆，记次大值为 $M_2$，次次大值为 $M_3$（若无 $M_3$，直接取完两堆），每堆取 $min(M_2,\lfloor\frac{S-2M_3}2\rfloor)$，取完后保证 新最大堆 ≤ 新总和的一半，复杂度 $O(ans\log X)$，最坏复杂度 $O(X\log X)$
