# 前置知识
## 线性表出
一个向量如果可以被其他向量通过运算表示，那么称这个向量可以被线性表出。

例如2维平面上的任意向量，都可以被(0,1),(1,0)这组向量通过矢量合成与放缩表示

在XCPC中，通常是一个数可以被其他数通过异或表示
## 线性相关&线性无关
如果向量组中有一个向量可以被向量组其他向量线性表出，那么称这个向量组线性相关，否则线性无关
## 极大线性无关组
线性相关可以理解为有多余的向量，删完之后，就是极大线性无关组
## 线性基
向量组的极大线性无关组就是线性基，简称基

一个向量组可以对应多个基，因此需要构造出一种合法基
### 性质
1. 原序列里的任意一个数都可以通过线性基中的一些数异或得到
2. 线性基里的任意一些数异或起来都不能得到 0
3. 每个序列的线性基的元素数量唯一，在性质一的前提下，有最少的数

by the way，序列里的元素相互异或一下，它的线性基不变

# vector实现
构造基的过程是向已经得到的基，插入新元素，如果可以被基线性表出，则舍去，否则保留。空集的基是空集
```cpp
vector<ull> B;
void insert(ull x)
{
    for (auto b : B)
        x = min(x, b ^ x);
    if (x)
    {
        for (auto& b : B)
            b = min(b, b ^ x);
        B.push_back(x);
    }
}
```
这种构造得到的基，有2个性质：

1. 每个元素最高有效位各不相同

2. 如果某一位是一个元素的最高有效位，则其他元素在这一位均为 0
## 性质证明
归纳证明：空集显然

插入新元素时，如果x变小，那么x在b的最高有效位是1，此时必须异或，否则之后没机会消掉x的这一位。

如果x在b的最高有效位是0，那么x要变大

一路操作下去，如果x是0，那么可以被线性表出，否则，新元素的最后结果的最高有效位一定和基中每个元素的最高有效位不同，即性质1。而且得到了x与集合内其他元素的组合异或最小值。

而第二个循环则是维护了性质2
# K-th异或和
对基按照大小排序，即可根据k的二进制选择基求出第k小的异或和

不过还要考虑 0 的情况，如果构造的线性基比原集合小，说明原集合可以异或出 0 ，那么我们需要将 k 减去 1 。

如果要判断输入合法性，不同的异或和个数有 $2^{|B|}-(!zero)$
```cpp
void init2()
{
    ranges::sort(B);
}
ll kthxorsum(ull k)
{
    k-=(B.size()<insertElementCount);
    if(k>=(1ll<<B.size()))
        return -1;
    ll ans=0;
    for(;k;k&=(k-1))
        ans^=B[countr_zero(k)];
    return ans;
}
```
# 异或和rank
n个元素有 $2^n$ 个子集，根据二进制反向处理

但是集合如果是线性相关的，就会有重复值，这里有个结论：

每个值都出现一样的次数，且这个次数 $=2^{n-|B|}$
```cpp
void init2()
{
    ranges::sort(B);
}
ull askrank(ull x)
{
    ull ans=0;
    for (uint32_t i=0;i<B.size();++i)
        if (x&(1ULL<<(bit_width(B[i])-1)))
            ans+=1<<i;
    return ans*fastpower(2,insertElementCount-B.size())+1;
}
```

```cpp
long long fast_power(long long base,long long exponent)
{
    long long result=1;
    for(;exponent>0;exponent>>=1)
    {
        if(exponent&1)
            result=result*base;
        base=base*base;
    }
    return result;
}
struct linearBasis
{

                ans+=1<<i;
        return ans*fastpower(2,insertElementCount-B.size())+1;

    bool check(ull x)
    {
        for (auto b : B)
            x = min(x, b ^ x);
```

# 1 到 n 的路径的最小异或和
所有路径异或和，都可以由任意一条 1 到 n 路径的异或和与图中的一些环的异或和来组合得到。

从1先走到一个环，绕环一次，再原路返回，即可得到环的异或和，来路的异或和被抵消了。

A，B是两条1到n的路径，A，B组成环，如果A更优，走B前绕这个环一圈即可得到A的结果，所以是任意一条路径。

dfs搜出部分环，线性基自己可以解决环套环

最小异或和前面介绍过了
```cpp
typedef unsigned long long ull;
struct linearBasis
{
    vector<ull> B;

    {
        for (auto b : B)
            x = min(x, b ^ x);
        for (auto& b : B)
            b = min(b, b ^ x);
        if (x)
            B.push_back(x);
    }
    ull minXorSumWithX(ull x)
```

# 前缀线形基
如果一个区间问题能用区间上的数构建出的线形基解决，那么就可以使用前缀线形基

这里不能使用线段树，因为会TLE，不能使用稀疏表和猫树，因为会MLE

如果是在线问题，为每个前缀维护一个线形基；如果是离线问题，只需将询问按右端点升序排列，即可只需要维护一个线形基

相比于基础线形基，每个基向量多存储一个值，表示它在原序列中的位置。插入数 x 时，从高位到低位考虑，贪心保留位置更靠近询问右端点的基向量。查询时只查询位置大于等于询问左端点的基向量

不变量：对前缀 r 的基底 B，若 B[i].pos < l，则 [l, r] 内任何子集异或都无法产生最高位为 i 的值。

证明（反证）：若存在子集 S ⊆ [l, r] 异或后最高位为 i，则插入 S 中下标最大的元素时（其 pos ≥ l），该插入路径上必然某次会把 pos ≥ l 的值放到 B[i] 位置（贪心 swap 保证留下的是 pos 最大的）。这与 B[i].pos < l 矛盾。

前缀线形基只能实现为上三角型（高位基向量可能有低位1），对于行最简型才能求的问题（求第K小，求合法值Rank），需要先提取出大于等于询问左端点的基向量，再 $O(B^2)$ 用低位基向量消高位基向量的低位1，这样操作之后的线形基才是行最简型

>前文介绍的vector实现是行最简型，不能直接搬过来用

下面板子同时提供了vector实现和array实现，离线查询接口和在线查询接口

```cpp
namespace PrefixLinearBasisVector
{
    struct LinearBasisVector
    {
        struct node { uint32_t val,pos; };
        vector<node> B;
        void insert(uint32_t val,uint32_t pos)
        {
            if(!val)
                return;
            for(auto &[b,bpos]:B)
            {
                if(bit_width(b)>bit_width(val))
                    continue;
                if(bit_width(b)<bit_width(val))
                    break;
                if(pos>bpos)
                {
                    swap(b,val);
                    swap(bpos,pos);
                }
                val^=b;
                if(!val)
                    return;
            }
            B.insert(ranges::lower_bound(B,val,greater{},&node::val),{val,pos});
        }
        uint32_t maxXorSum(uint32_t l)
        {
            uint32_t res=0;
            for (auto [b,pos] : B)
                if(pos>=l)
                    res = max(res, b ^ res);
            return res;
        }
    }plb[N];
    void init(uint32_t n)
    {
        for(uint32_t i=1;i<=n;++i)
        {
            plb[i]=plb[i-1];
            plb[i].insert(a[i],i);
        }
    }
    uint32_t maxXorSum(uint32_t l,uint32_t r)
    {
        return plb[r].maxXorSum(l);
    }
}
const uint32_t A = 1e6 + 1,BitA=bit_width(A);
namespace PrefixLinearBasisArray
{
    struct LinearBasisArray
    {
        struct node { uint32_t val,pos; }B[BitA];
        void insert(uint32_t val,uint32_t pos)
        {
            while(val)
            {
                auto b=&B[bit_width(val)-1];
                if(!b->val)
                {
                    *b={val,pos};
                    return;
                }
                if(b->pos<pos)
                {
                    swap(b->val,val);
                    swap(b->pos,pos);
                }
                val^=b->val;
            }
        }
        uint32_t maxXorSum(uint32_t l)
        {
            uint32_t res=0;
            for(int i=BitA-1;i>=0;--i)
                if(B[i].pos >= l)
                    res = max(res,res^B[i].val);
            return res;
        }
    }plb[N];
    void init(uint32_t n)
    {
        for(uint32_t i=1;i<=n;++i)
        {
            plb[i]=plb[i-1];
            plb[i].insert(a[i],i);
        }
    }
    uint32_t maxXorSum(uint32_t l,uint32_t r)
    {
        return plb[r].maxXorSum(l);
    }
}
```
