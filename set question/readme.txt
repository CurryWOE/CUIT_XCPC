出题流程：
1.想题目idea
2.想正解
3.用管理员账号登录cuitoj
4.右上角admin-management
5.左侧problem-create problem
6.填写：题号，题名，描述，输入描述，输出描述，时间限制，空间限制，样例的输入输出，提示（可选），spj（可选）
赛制（ACM/OI），测试数据，IO模式（标准输入/文件读写），其他不用管。
7.测试数据生成见input.cpp和output.cpp，分别生成测试数据的输入输出
8.input.cpp改动数据生成，output.cpp把标程放进大括号内即可
9.将测试数据（包括输入输出）打包成zip格式
10.cuitoj-choose file-刚刚打包的文件
11.save
